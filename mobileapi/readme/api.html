手机接口

数据格式约定

返回数据为json格式，code为200表示数据正常，data为返回的数据 

{code: 200, data: {}}

返回值说明

获取数据

        //有内容直接返回数组，无内容返回空数组[],hasmore为是否有下一页
        {"code":200,"hasmore":true,"data":{"goods_list":[]}}

操作成功

        //操作成返回"true"
        {"code":200,"data":{"true"}}

错误

        //出现错误返回error，error内为错误信息
        {"code":301,"error":"参数错误"}

根据分类获取商品列表

数据接口(GET)

category.php

请求参数

  • id 筛选条件 分类ID
  • brand 筛选条件 品牌ID
  • price_max 筛选条件 上限价格
  • price_min 筛选条件 下限价格
  • page 当前页数
  • page_size 每页显示多少条数据

返回数据

  • goods_list 商品数组(array)
    • goods_id 商品ID
    • goods_name 商品名称
    • goods_thumb 商品图片
    • shop_price 商品价
    • market_price 市场价
    • promote_price 促销价
    • is_new 1 表示 新品
    • is_best 1 表示 精品
    • is_hot 1 表示 热销
